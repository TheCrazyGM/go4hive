# Tech Stack

## Backend
- **Language:** Python 3.14+
- **Framework:** Django 6.x
- **SDK:** `hive-nectar` (the modern successor to `beem`)
- **Package Manager:** `uv`
- **Environment Management:** `.venv` (managed by `uv`)

## Frontend
- **Templating:** Django Templates (Server-Side Rendering)
- **Styling:** Vanilla CSS (Custom Terminal Theme)
- **Fonts:** Monospace (e.g., Courier New, VT323, Fira Code)
- **Icons:** Unicode characters and simple ASCII art

## Tools & Standards
- **Linter/Formatter:** `ruff`
- **Type Checking:** `ty` (via `uv`)
- **Testing:** `pytest` / Django Test Runner
